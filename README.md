# 22framboise_project

## Docker

Aller dans le dossier /cloud puis : 

- Pour démarrer : 
``docker-compose up``


- Pour stopper : 
``docker-compose down``

- Si l'un des Dockerfiles a été modifié :
``docker-compose build``
